# Commands

Slash command definitions reusable across AI tools.

## Available Commands

| Command | Description |
|---------|-------------|
| [/pai](pai.md) | Manage PAI - sync learnings, export skills, update preferences |

## Structure

Each command file should include:
- **Name** - The command name (e.g., `/review`)
- **Description** - What it does
- **Arguments** - Optional arguments it accepts
- **Workflow** - Steps the command executes

## Adding Commands

Create a new `[command-name].md` file in this directory.
